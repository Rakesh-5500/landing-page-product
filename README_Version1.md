# ProductX - Landing Page with Authentication

A modern, responsive landing page with animated light/dark theme switching and complete authentication system including Sign Up, Sign In, and Forgot Password functionality.

## Features

✨ **Modern Design**
- Responsive layout for all devices
- Beautiful gradient effects
- Smooth animations and transitions
- Clean typography and spacing

🌙 **Theme Mode**
- Animated light/dark theme toggle
- Smooth transitions between themes
- User preference persistence
- System preference detection

🔐 **Authentication System**
- Sign Up with email validation
- Sign In with remember me option
- Forgot Password functionality
- Local storage for user data (demo only)

📱 **Responsive Design**
- Mobile-first approach
- Desktop, tablet, and mobile optimization
- Touch-friendly buttons and inputs

🚀 **Performance**
- Fast loading
- Optimized animations
- No external dependencies (except Font Awesome)
